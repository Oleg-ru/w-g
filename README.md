## Как внести изменения

Все правки вносятся в основной репозиторий `w-g`. Чтобы обновить этот репозиторий, выполните следующие шаги:

1. Перейдите в корень основного репозитория
2. Внесите необходимые изменения в папку проекта
3. Зафиксируйте изменения:
bash git add . git commit -m "Описание изменений"
4. Отправьте изменения в этот репозиторий с помощью команды:
bash git subtree push --prefix=practice/propmise/04-web-meteo weather-deploy main
